    ● Function template 和class template 的设计和运用

    ● Exception handling(异常处理)与执行期型别鉴定（Run-Time Type Identification）

    此外，书后的两份附录极具价值。附录A提供每章最后所列的练习题的完整解答和详细说明。附录B提供一份泛型演算法快速参考手册（含运用实例）。
